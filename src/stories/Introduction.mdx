# Frontend S1 Design System

Welcome to the **Frontend S1** design system built with React, TypeScript, and Chakra UI v3.

## 🚀 Features

- **Modern React 19** with TypeScript for type safety
- **Chakra UI v3** with custom theme system
- **Vite** for fast development and building
- **ESLint** for code quality
- **Error Boundaries** for graceful error handling
- **Responsive Design** with mobile-first approach

## 📁 Project Structure

```
src/
├── components/     # Reusable UI components
├── hooks/         # Custom React hooks
├── types/         # TypeScript type definitions
├── utils/         # Utility functions
├── stories/       # Storybook documentation
├── App.tsx        # Main application component
├── main.tsx       # Application entry point
└── theme.ts       # Chakra UI theme configuration
```

## 🎨 Design System

Our design system includes:

- **Typography Scale** - Consistent heading and text sizes
- **Color Palette** - Semantic colors for different states
- **Button System** - Multiple variants, sizes, and color schemes
- **Spacing Scale** - Consistent spacing throughout the app
- **Component Library** - Reusable, documented components

## 🛠️ Development

### Running Storybook

```bash
yarn storybook
```

### Building the Project

```bash
yarn build
```

### Linting

```bash
yarn lint
```

## 📖 Component Documentation

Each component in this design system includes:

- **Interactive controls** for testing different props
- **Multiple stories** showing various states and use cases
- **TypeScript documentation** with prop descriptions
- **Accessibility considerations** where applicable

Navigate through the sidebar to explore individual components and their documentation.

## 🔧 Customization

The theme system is highly customizable. Check out the `src/theme.ts` file to see how colors, typography, spacing, and component recipes are defined.

## 🧪 Testing Components

Use the **Controls** panel at the bottom of each story to interact with component properties and see how they behave in different states.
